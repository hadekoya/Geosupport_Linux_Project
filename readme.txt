# Install Geosupport on Linux



# Building from Source

## Setting Up Envrionment Variables




Brief Run Through for Creating Your Own C Application That Calls Linux Geosupport 

1)	Install Linux Geosupport 
2)	Read linux_geosupport_doc.txt 
3)	Modify the c_client_simple_make.mak to point to the installation location (see comments).
4)	Run the c_client_simple_make.mak to test this sample. 
5)	If everything works correctly, modify the make file for your C application code 
6)	Use the information provides in linux_geosupport_doc.txt to reference your installation of Linux Geosupport for your application to use.
